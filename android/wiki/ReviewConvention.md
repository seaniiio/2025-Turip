# 코드 리뷰 컨벤션

- Merge는 1명 이상의 인원에게 승인(approve)을 받는다.
- RCA룰을 통해 Prefix를 적고, 코드 리뷰 반영의 우선순위를 표시한다.
    - r : 무조건 반영해주세요
    - c : 사소한 의견
    - a : 넘어가도 되는 사항
    
    **(Approve(승인), Comment(피드백 및 토론), Request Changes(수정 요청))**
